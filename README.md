# Simple Proxy Server (C++)

## Description
This project implements a basic proxy server in C++ that
accepts incoming connections and forwards traffic to target servers.
It demonstrates understanding of sockets, networking, and C++ I/O.

## Features
- TCP proxy
- configurable IP/Port
- simple command-line input
- sample client/server tests

## How to build
```bash
g++ copymain.cpp -o proxy
